# Birthday Reminder App

A beautiful and easy-to-use React Native application built with Expo to help you keep track of birthdays and name days. Never miss a celebration again!

## Features

- **Track Birthdays & Name Days**: Easily add and manage important dates for your friends and family.
- **Daily Celebrations**: See who is celebrating today at a glance.
- **Smart Notifications**: Get timely reminders so you're always prepared.
- **Custom Event Types**: Support for custom celebration types beyond just birthdays.
- **Modern UI**: Clean, intuitive interface designed for iOS and Android.

## Tech Stack

- **Framework**: [React Native](https://reactnative.dev/) with [Expo](https://expo.dev/)
- **Routing**: [Expo Router](https://docs.expo.dev/router/introduction/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Icons**: [Lucide React Native](https://lucide.dev/guide/packages/lucide-react-native)
- **State Management**: React Context API

## Getting Started

### Prerequisites

- Node.js (LTS version recommended)
- npm or yarn
- [Expo Go](https://expo.dev/client) app on your mobile device (for testing)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/birthday-reminder.git
   ```

2. Navigate to the project directory:
   ```bash
   cd birthday-reminder
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

Start the development server:

```bash
npm run dev
```

Scan the QR code with the Expo Go app (Android) or Camera app (iOS) to run the application on your device.

## Project Structure

- `/app`: Application screens and navigation (Expo Router).
- `/components`: Reusable UI components.
- `/contexts`: Global state management.
- `/hooks`: Custom React hooks.
- `/data`: Static data (e.g., name days).
- `/types`: TypeScript type definitions.
